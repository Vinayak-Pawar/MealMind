# MealMind

A smart meal planning and nutrition tracking application.

## Features

- Meal planning and recipe suggestions
- Nutrition tracking
- Dietary preference management
- Shopping list generation

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
pip install -r requirements.txt
```

### Usage

```bash
python app.py
```
